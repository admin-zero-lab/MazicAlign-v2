### 주요 기능 (v2 가 추가한 것)

- **다중 STL** 동시 편집 + IndexedDB 영속화 + Undo/Redo (callback 기반)
- **Gizmo** — Babylon PositionGizmo/RotationGizmo/ScaleGizmo 직접 인스턴스
- **자동 서포트** — 격자 샘플링 + 모델 아래에서 위로 ray (auto-generate.ts)
- **수동 서포트** — 모델 표면 클릭 = 추가 / 기둥 클릭 = 선택 / Delete = 제거
- **Bridge** — 두 모델 / 두 Bridge 표면을 잇는 cross-brace
  - 직선 + 곡선 (Catmull-Rom + 변곡점 가변 길이)
  - A/B 끝점 sphere + normal 저장 → 표면 밖 lift / 안쪽 push 분리
  - 부모 ↔ child follow (t 비율 저장)
  - 변곡점 tube 위 더블클릭 = 추가 / Delete = 제거
- **STL transform 동기** — STL 이동/회전/스케일 시 부착된 모든 서포트도 함께 변환
- **슬라이스 미리보기** — Z 평면 단면 + 변곡점/메시 교차 라인 + 1bpp 마스크
- **PNG/ZIP 내보내기** — 의존성 없이 자체 인코더
- **CTB 인코더** — v3/v4 (현재 비활성, 사용자 검증 대기)
- **PrinterProfile** — 빌드볼륨/해상도 단일 출처
- **출력 추정** — 부피 → 레진 ml + 시간

## 5. 핵심 변경 요약

옛 popcaron/MazicAlign 은 **로그인 → 백엔드 API → STL 업로드 → 슬라이서** 의 단순 구조였음. 이번 작업이 추가한 것:

1. **백엔드 의존성 없는 v2 모듈** — IndexedDB 영속화로 인증/네트워크 불필요
2. **자동/수동 서포트** + **Bridge** 라는 새 컨셉의 cross-brace 서포트
3. **자체 슬라이서 출력 파이프라인** — PNG/ZIP/CTB 모두 의존성 없이 구현
4. **사용자 인터랙션** — Gizmo, 더블클릭, 바닥면 붙이기, viewport overlay 우측 정렬
5. **데이터 모델** — `SupportPointV2`, `TransformV2`, `STLFileV2`, IndexedDB v4 마이그레이션
